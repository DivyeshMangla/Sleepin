# Sleepin

A minimalist, high-reliability wake-up application designed to enforce consistent early rising through unskippable alarms, cognitive tasks, motion checks, and sensor-based verification. Built for experimentation, learning, and practical exploration of mobile alarm-system mechanics using Flutter.
